# Simple Interest Calculator

A simple command-line tool written in Shell/Python to calculate simple interest based on principal, annual rate of interest, and time period in years.

## Formula
The formula used for the calculation is:
**SI = P * R * T**

Where:
* **P** = Principal amount
* **R** = Annual rate of interest
* **T** = Time period in years

## Usage
Input the following values when prompted:
1.  `principal`: the amount originally invested/borrowed.
2.  `interest rate`: the annual rate (e.g., 5 for 5%).
3.  `time`: the number of years.

## Output
The script will output the total Simple Interest earned/owed.
